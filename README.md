# 🌤️ Weather App

A responsive Weather App that provides real-time weather data and a 3-day forecast using API integration.

## 🚀 Features

- 🌍 Real-time weather data based on city search
- 📅 3-day weather forecast
- 📍 Geolocation support (detect user location)
- 🎨 Dynamic background based on weather conditions
- ⚡ Loading state for better user experience
- ❌ Custom error handling (no browser alerts)
- 📱 Fully responsive design

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript (Vanilla JS)
- Weather API
